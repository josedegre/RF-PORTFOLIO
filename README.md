# RF & Microwave Engineering Portfolio

Collection of RF, microwave and antenna design projects covering simulation, circuit design, and experimental validation.

## Author
José María De Gregorio Estúñiga  
Telecommunications Engineering – RF & Microwave specialization

## Overview

This repository showcases hands-on experience in:

- Antenna design (patch, arrays, reflectors, circular polarization)
- Phased array analysis and beamforming
- RF front-end components (LNA, filters, combiners, phase shifters)
- Electromagnetic simulation (CST)
- Circuit design (Keysight ADS)
- Signal processing and modeling (MATLAB)
- Experimental validation (VNA, anechoic chamber)
