# Waveguide Bandpass Filter Design

## Description
Design and analysis of a waveguide bandpass filter in X-band using cavity resonators and inductive irises.

## Specifications
- Passband: 9.8–10.2 GHz
- Return loss: ≥ 18 dB
- Insertion loss: ≤ 0.1 dB
- Stopband attenuation: up to 40 dB

## Methodology
- Lumped-element filter synthesis
- Transformation to λ/2 resonators
- Waveguide implementation using TE101 cavities
- Validation with Mode-Matching and CST Microwave Studio

## Tools
- CST Studio Suite
- MATLAB
